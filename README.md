# [심층 분석] 라스트 워: 서바이벌(Last War: Survival) 전 기능 리포트

본 문서는 `Last War: Survival`의 핵심 시스템을 9개 카테고리로 나눠 정리한 통합 분석 보고서입니다.  
게임의 표면적 재미(슈팅형 연출) 뒤에 있는 실제 성장 구조(SLG 메커니즘)를 플레이 루프 중심으로 해설합니다.

## 보고서 목적

- 게임 내 UI와 시스템을 기능 단위로 재구성
- 숨겨진 성장 병목(시간/재화/연맹 의존)을 파악
- 일일/주간 이벤트가 유저 행동을 어떻게 통제하는지 설명
- 과금/협력/전투가 맞물리는 경제 루프를 가시화

## 1) 기지 건설 및 인프라 (Base Construction)

기지 레벨은 플레이어의 실질적인 계정 단계이자, 군사/연구/콘텐츠 접근 권한을 결정하는 관문입니다.

- **본부 (Headquarters)**: 전체 건물 상한 레벨과 고티어 병사 해금 조건을 규정하는 최상위 허브
- **자원 생산 시설**
  - 철광산 / 식량 농장 / 에너지 발전소: 시간당 자원 생산
  - 레벨 상승 시 생산량, 저장량, 성장 효율 동시 증가
- **창고 (Warehouse)**: 약탈 피해를 줄이는 보호 자원 상한 제공
- **병원 (Hospital)**: 부상병을 치료해 전투 손실을 회수, 용량 초과 시 영구 손실 발생
- **병영 (Barracks)**: 전차/항공기/미사일 병종별 훈련 허브
- **벽 (Wall)**: 방어 지속성의 핵심. 내구도 0 시 기지 강제 이전 트리거
- **데코레이션/스킨**: 단순 미관을 넘어 일부 유료 스킨이 전투 버프 제공

## 2) 군사 및 병력 시스템 (Military & Units)

전투 결과는 병력 총량보다 **티어 차이 + 상성 + 영웅 세팅**에 의해 크게 갈립니다.

- **병사 티어 (T1 ~ T10)**: 본부/연구 진척에 따라 해금
- **핵심 상성**
  - 전차 > 미사일차
  - 미사일차 > 항공기
  - 항공기 > 전차
- **차고 (Garage)**: 최대 4개 부대 운용, 각 부대당 영웅 5명 배치
- **부대 리모델링**: 외형 변경 + 추가 능력치로 미세 최적화 가능

## 3) 영웅 시스템 (Heroes)

실전 전투력 대부분이 영웅 투자 상태에서 결정됩니다.

- **등급 구조**: UR / SSR / SR (메타는 UR 중심으로 수렴)
- **육성 축**
  - 레벨업: 경험치 아이템 기반 수치 상승
  - 성급 강화(Tier): 조각 소모, 스킬/스탯 도약 구간
  - 스킬 강화: 4개 고유 스킬을 메달로 성장
- **영웅 장비**
  - 장착 슬롯: 무기 / 갑옷 / 레이더 / 칩
  - 강화/제련: 레벨 및 품질 확장
  - 전용 장비: 특정 영웅에 고유 시너지 부여

## 4) 과학 기술 연구 (Tech Center)

기술 센터는 성장 속도와 최종 전투력의 장기 기울기를 만드는 시스템입니다.

- **경제 기술**: 건설 속도, 채집 효율 등 시간 대비 수익 향상
- **군사 기술**: 공격/방어/훈련 효율 및 고티어 병사 개방
- **영웅 기술**: 영웅 기반 전력 증폭
- **특수 기술**: VS 점수 가속, 방어 보정 등 시즌/이벤트 실전형 옵션

## 5) 월드맵 콘텐츠 (World Map / Wilderness)

기지 외부 필드는 실시간 성장 재료를 확보하는 핵심 생산지이자 협동 전장의 시작점입니다.

- **좀비 처치**: 레벨별 전투로 자원/경험치 수급
- **자원 타일 채집**: 철/식량/에너지 확보
- **엘리트/보스**: 단독 처치 한계가 뚜렷해 집결(Rally) 유도
- **레이더 임무 (Radar Tasks)**: 일일 보상 파이프라인의 중심
- **비밀 임무 (Secret Tasks)**: 파견형 콘텐츠 + 약탈 리스크 내재

## 6) 연맹 시스템 (Alliance) - 실질적 핵심

이 게임은 구조적으로 연맹 참여를 강제에 가깝게 요구합니다.

- **연맹 도움**: 건설/연구 대기시간 단축
- **연맹 기술**: 집단 버프를 통한 전 서버 경쟁력 확보
- **연맹 상점**: 기여도 소비형 핵심 물자 공급
- **연맹 대결 (VS Alliance)**: 요일별 과제 수행으로 승패 결정
- **집결 전투**: 보스/유저 기지 대상 협동 공격의 기본 전술

## 7) 시즌 및 정기 이벤트

반복 피로를 줄이면서도 접속 루틴과 과금 동기를 강화하는 순환 설계입니다.

- **SvS (서버 vs 서버)**: 서버 단위 대전쟁, 수도 점령과 권력 구조 형성
- **군비 확장 (Arms Race)**: 특정 행동 반복을 보상하는 상시 루프
- **특수 작전 (Special Ops)**: 광고형 미니게임 접점 콘텐츠
- **무기 공장 / 사막 전장**: 시간 제한 PvP 인스턴스

## 8) 편의 및 과금 시스템

편의 기능은 대부분 성장 속도 또는 기회비용 절감 형태로 과금과 연결됩니다.

- **VIP 시스템**: 추가 건설열, 생산량 보정 등 장기 효율 특화
- **가속 아이템**: 건설/연구/훈련 시간 즉시 절감
- **시즌 패스**: 월간형 누적 보상 설계
- **드론 (Drone)**: 별도 성장축을 가진 전투 보조 시스템

## 9) 경제 메커니즘 (Resource Loop)

`생산/채집 -> 성장 투자 -> 전투 성과 -> 희귀 재화 회수 -> 재투자`가 기본 골격입니다.

1. 기지 내 생산 + 월드맵 채집으로 기초 자원 확보
2. 건설/연구/육성에 재화를 투입해 전투력 상승
3. 이벤트/PvP 참여로 다이아/영웅 조각 등 희소 재화 획득
4. 부족 자원은 연맹 협력 또는 과금으로 보충

## 구조적 결론

`Last War: Survival`은 외형상 캐주얼 슈팅 요소를 전면에 두지만, 내부는 다음의 정교한 SLG 프레임으로 작동합니다.

- 영웅 육성 중심의 전투력 압축
- 본부/연구 기반의 단계적 계정 락 해제
- 연맹 중심의 시간 단축 및 전쟁 동원
- 서버 간 시즌 경쟁으로 장기 체류 동기 강화
- 요일별 VS 미션을 통한 플레이 루틴 고정

즉, 이 게임의 핵심은 단순 전투가 아니라 **시간 관리, 집단 협업, 재화 효율 최적화**에 있습니다.
